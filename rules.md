# 鹿児島弁変換規則表 v0

標準語を鹿児島弁（薩隅方言）に変換するための自作規則表。
一般に知られる薩隅方言の特徴を整理したもので、辞書サイトからの転載ではない。

⚠️ **この表はネイティブチェック前の草稿**。地域（薩摩／大隅／北薩など）や世代で
言い方が変わるため、地元の話者に確認してから確定させること。

## F. 実録「アルティメット鹿児島弁」から採取した規則

2018年8月22日に鹿児島市で起きたパトカー自損事故の目撃者インタビュー
（X/旧Twitterで「アルティメット鹿児島弁」として話題）から採取した、
**実際のネイティブ発話に基づく規則**。A〜Eの机上規則と違い、生きた話し言葉の証拠。

| # | 標準語 | 鹿児島弁 | 例（実録より） |
|---|--------|----------|-----------------|
| F1 | 〜して（ですね） | 〜しっせえ（よ） | わっぜえ音がしっせえよ |
| F2 | 〜かと思って（ですね） | 〜か思っせえよ | 今朝ん台風か思っせえよ |
| F3 | 〜してみたら（ですね） | 〜してみっせな | 見てみっせな |
| F4 | 〜していた（発見） | 〜ちょら | 警察がとまっちょら |
| F5 | 〜した限りでは | 〜たぎいな | そこら辺見たぎいな |
| F6 | 私（女性の自称） | あたい | あたいは今朝ん… |
| F7 | どうしてかと | ないごてけ | ないごてけ思って |
| F8 | あれが | あいが | あいがと左角ぶつけっせえよ |
| F9 | 〜さん（人・組織を指して） | 〜どん | 警察どん |
| F10 | 〜してはいけないでしょう | 〜さあよかなかど | 事故さあよかなかど |

**考察メモ**: 語尾「〜しっせえ」「〜ちょら」は辞書ベースの規則表（A〜E）には
出てこなかった。机上の規則だけでは実際の話し言葉を再現できない証拠であり、
実録データの重要性を示している（記事のネタになる）。

## 適用方針

1. まず D（語彙）と C（形容詞）を置換する
2. 次に A（文末）を書き換える。文末が一番「らしさ」が出る
3. B・E は読みやすさと相談して**控えめに**使う（全部促音化すると読めなくなる）
4. 固有名詞・数字・専門用語はそのまま残す
